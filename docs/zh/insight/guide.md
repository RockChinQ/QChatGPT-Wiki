# 阅读指引

## 部署板块

部署使用本项目请严格按照以下三步进行，其中每一步可以选择不同的方式进行。

   - 阅读 部署 LangBot 页，选择部署方式
   - 阅读 部署消息平台页，选择要使用的消息平台
   - 阅读 填写配置信息页，根据情况填写配置
## 使用板块
   - 阅读命令语法了解机器人可用命令，可以在对话中实现切换预设、切换模型等
   - 阅读插件列表，获得插件，增强功能
   - 阅读常见问题，可有助于解决部署和使用时的问题
## 配置板块
   - 功能配置，具体解释所有的配置信息，实现更多自定义需求
   - 元数据配置，控制敏感词、模型列表等
